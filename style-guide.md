# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Soft orange: hsl(35, 77%, 62%)

Off-white: hsl(36, 100%, 99%)
Grayish blue: hsl(233, 8%, 79%)

Very dark blue: hsl(240, 100%, 5%)
